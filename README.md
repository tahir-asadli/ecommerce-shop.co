# Shop.co

A modern e-commerce clothing store built with HTML, Tailwind CSS, and vanilla JavaScript.

## Preview

[Preview](https://e-commerce-shop-co.netlify.app/)

## Pages

- [**Home**](https://e-commerce-shop-co.netlify.app/) — Hero section, featured categories, new arrivals, and customer reviews
- [**Category**](https://e-commerce-shop-co.netlify.app/category.html) — Browse products by category with filters
- [**Product**](https://e-commerce-shop-co.netlify.app/product.html) — Product detail page with image gallery and size selector
- [**Cart**](https://e-commerce-shop-co.netlify.app/cart.html) — Shopping cart with order summary

## Tech Stack

- HTML5
- [Tailwind CSS v4](https://tailwindcss.com/)
- Vanilla JavaScript
- [Splide.js](https://splidejs.com/) — carousel/slider

## Getting Started

Install dependencies:

```bash
npm install
```

Build the CSS:

```bash
make tailwind-build
```

Or watch for changes during development:

```bash
make tailwind-watch
```
